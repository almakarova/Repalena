1) Закоммиить все, что ты написала! Пока в репозитории только текстовые файлы, так что я не могу ничего проверить.
  - убедиться, что у тебя папка с проектом под контролем гита (ты должна была git clone запускать)
  - дальше тебе понадобяся команды git add . 
  - закомитить git commit -m 'my comment here'
  - чтобы подсосать измения удаленно git pull --rebase (с rebase лучше, так история наших с тобой изменеий будет линейна)
  - ну и git push

2) Давай еще одну стандарную программку - калькулятор. Для ввода будем использовать параметры main(String[] args). С массивами
ты уже знакома, так что теперь что за аргументы:
  a = args[0] - '/','*','-','+'  
  b = args[1] - Integer
  c = args[2] - Integer 
Задача - произвести действие а над числами b и c и вывести результат. Тебе понадобится конструкция if, String -> Integer
и научиться в идее прописывать параметры программы, arg-и в общем.
